# ServeNow 🚀
Instant Personal Care & Home Services

ServeNow is a hyperlocal service companion app that brings professional care for your clothes, shoes, and accessories right to your doorstep. With pickups in as little as 15–30 minutes, ServeNow is designed for speed, convenience, and accessibility.

**[Live Demo](https://amisha-srivastavaa.github.io/ServeNow/)**

## ✨ Key Features

### 🛍️ Comprehensive Marketplace
- **Ironing, Dry Cleaning & Laundry:** Professional fabric care with express options.
- **Shoe & Bag Repair:** Bring your favorite accessories back to life.
- **Tailoring & Stitching:** Custom fits and rapid alterations.

### 🤖 AI-Powered Style Assistant
- **Smart Fit:** Get personalized measurement recommendations.
- **Style Quiz:** Discover your unique fashion vibe (Minimal, Bold, Traditional, or Sporty).
- **Style Feed:** Explore community posts, fashion tips, and styling trends.

### 👵 Senior Mode
ServeNow is built with accessibility in mind. Toggle **Senior Mode** for:
- **Simplified Navigation:** Larger icons and straightforward workflows.
- **Voice Assistant:** Built-in voice commands with multilingual support (English and Hindi).
- **One-Tap Help:** Direct connections to support agents.

### 🛵 Live Tracking & Real-Time Updates
- Track your order live on the map from pickup to processing to delivery.
- View precise ETAs and status updates.
- Instantly connect with your service provider.

### 🏪 Provider Dashboard
- Built-in portal for service providers.
- Toggle Online/Offline status.
- Accept or decline incoming orders in real-time.
- View daily earnings and active order stats.

## 🛠️ Technology Stack
- **HTML5 & CSS3:** Custom, modern, and responsive UI components.
- **Vanilla JavaScript:** Fast, lightweight, module-free frontend architecture.
- **Vite:** Next-generation frontend tooling for rapid development and optimized builds.
- **GitHub Pages:** Automated deployment directly from the `gh-pages` branch.

## 🚀 Getting Started

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/amisha-srivastavaa/ServeNow.git
   cd ServeNow
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open your browser and navigate to `http://localhost:5173/ServeNow/`.

## 📦 Deployment
This project is configured to deploy to GitHub Pages automatically.

To build and deploy the application manually:
```bash
npm run deploy
```
*Note: This runs Vite's build process and pushes the `dist` directory to the `gh-pages` branch.*
